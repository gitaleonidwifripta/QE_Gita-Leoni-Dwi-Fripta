Web Programming
Web Programming atau pemrograman web adalah sebuah bidang dalam teknologi informasi yang berkaitan dengan pengembangan aplikasi 
web, website, dan aplikasi mobile yang berbasis web. Pemrograman web melibatkan penggunaan bahasa pemrograman dan teknologi web seperti 
HTML, CSS, JavaScript, dan berbagai macam bahasa pemrograman server-side seperti PHP, Java, Ruby, Python, dan sebagainya.

ccs adalah CSS adalah bahasa stylesheet yang digunakan untuk mengatur tampilan atau gaya halaman web, seperti font, warna, ukuran, jarak antar elemen, 
dan tata letak halaman. Dengan menggunakan CSS, pengembang web dapat memisahkan konten dari tampilan dan gaya halaman, 
sehingga memungkinkan untuk mengubah tampilan halaman secara terpusat dan efisien, tanpa perlu mengubah konten HTML-nya

Boostrap 
Bootstrap adalah kerangka kerja (framework) front-end yang populer digunakan dalam pengembangan web. Bootstrap menyediakan kumpulan komponen antarmuka 
pengguna (UI) yang siap digunakan, seperti tombol, formulir, jumbotron, slider, navigasi, dan banyak lagi. Selain itu, Bootstrap juga menyediakan sistem 
grid yang fleksibel untuk membantu pengembang web dalam menata tata letak halaman secara responsif, sehingga halaman web dapat menyesuaikan tampilannya 
dengan baik pada berbagai perangkat dan ukuran layar yang berbeda. 

2. Backend
Backend adalah istilah yang digunakan untuk merujuk pada bagian dari aplikasi atau situs web yang tidak terlihat oleh pengguna akhir, 
tetapi berfungsi untuk mengatur dan memproses data. Backend biasanya terdiri dari server, database, dan aplikasi server yang bertanggung jawab untuk 
menangani permintaan dari klien dan memproses data sesuai dengan permintaan tersebut.

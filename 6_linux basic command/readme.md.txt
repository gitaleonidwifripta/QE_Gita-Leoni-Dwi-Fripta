Linux Basic Command
Linux Basic Command adalah sekumpulan perintah (command) yang biasa digunakan pada sistem 
operasi Linux dan berjalan melalui terminal atau command line. Beberapa perintah command tersebut antara lain:

1. ls - untuk menampilkan daftar file dan direktori pada lokasi yang aktif.
2. cd - untuk berpindah ke direktori lain.
3. mkdir - untuk membuat direktori baru.
4. touch - untuk membuat file kosong atau memperbarui timestamp file yang sudah ada.
5. cp - untuk menyalin file atau direktori dari satu lokasi ke lokasi lain.
6. mv - untuk memindahkan atau mengganti nama file atau direktori.
7. rm - untuk menghapus file atau direktori.
8. cat - untuk menampilkan isi file ke layar.
9. nano - untuk mengedit file teks.
10.grep - untuk mencari pola tertentu dalam file atau output dari command lain.
11.ps - untuk menampilkan daftar proses yang sedang berjalan di sistem.
12.top - untuk memonitor penggunaan CPU dan memori oleh proses-proses yang sedang berjalan di sistem.
13.chmod - untuk mengubah hak akses (permission) file atau direktori.
14.chown - untuk mengubah kepemilikan (ownership) file atau direktori.

Selain itu, ada banyak lagi command line pada Linux yang dapat digunakan untuk mengatur dan memanipulasi file, mengelola jaringan,
memonitoring sistem, dan lain sebagainya. 
Command line pada Linux juga memiliki banyak opsi (option) dan argumen yang dapat dikombinasikan untuk mencapai tujuan yang diinginkan.

Shell script pada Linux adalah sebuah skrip atau program yang ditulis dengan menggunakan bahasa pemrograman shell pada sistem operasi Linux.
Shell script dapat menjalankan serangkaian perintah atau command line secara otomatis, sehingga memudahkan pengguna dalam mengeksekusi beberapa command line secara bersamaan.

Dalam membuat shell script, kita dapat menggunakan teks editor seperti nano atau vi, lalu menyimpan file tersebut dengan ekstensi .sh. 
Kemudian, kita harus memberikan hak akses eksekusi pada file tersebut dengan menggunakan perintah chmod +x nama_file.sh.

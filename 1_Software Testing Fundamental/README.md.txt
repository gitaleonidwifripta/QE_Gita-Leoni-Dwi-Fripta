Software Testing fundamental & Software Testing as a Career
Software Testing fundamental
Pengertian Software Testing adalah  suatu proses menganalisis perangkat lunak untuk mendeteksi perbedaan antara kondisi yang ada dan yang dibutuhkan  seperti cacat dan untuk mengevaluasi fitur item perangkat lunak. 

Testing platform 
1.	Api 
Api Testing  adalah tipe software testing yang memvalidasi Api, terjadi pada Business Layer, Software Layer yang paling kritis karena menghubungkan Presentation Layer dan Database Layer. 
2.	Web
Pengujian Web secara sederhana memeriksa aplikasi web Anda untuk kemungkinan bug sebelum dibuat langsung atau sebelum kode dipindahkan ke lingkungan produksi.
3.	Mobile
Mobile application testing adalah software automation yang dikembangkan untuk deteksi perbedaan antara kondisi mobile app yang ada dengan kondisi yang tidak diinginkan (defects, errors, bugs) serta mengevaluasi fitur-fitur yang ada dalam mobile app
4.	Dekstop
Desktop adalah praktik pengujian perangkat lunak yang memeriksa fungsionalitas, keamanan, kegunaan, dan stabilitas aplikasi setelah diterapkan. Dalam pengujian aplikasi desktop, Anda harus memperhatikan pemasangan serta pengujian pemasangan untuk sepenuhnya memenuhi persyaratan pengujian aplikasi.

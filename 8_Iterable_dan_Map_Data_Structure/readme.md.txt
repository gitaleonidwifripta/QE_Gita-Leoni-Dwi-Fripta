Iterable & Map Data Structure
Iterable dan Map adalah dua jenis struktur data yang berbeda.

Iterable
Iterable adalah struktur data yang memungkinkan kita untuk melakukan iterasi atau perulangan pada setiap elemen yang terkandung di dalamnya. 
hirarchy interface :
1. iterable
2. collection
3. set
4. sorted set
5. navigable set
6. list
7. queue
8. deque
9. blocking deque
10.blocking queue
11. transfer queue

implementation
1. interfaces
2. hash table
3. resizable
4. tree
5. linked list
6. hash table + linked list




Map
Map adalah struktur data yang memungkinkan kita untuk menyimpan pasangan key-value (kunci-nilai).
Key adalah identifier yang unik yang digunakan untuk mengakses nilai yang terkait. Map juga dikenal sebagai dictionary di beberapa bahasa pemrograman
 implementasi map
HashMap
WeakHashMap
IdentityHashMap
LinkedHashMap
EnumHashMap
